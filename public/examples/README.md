# Examples Folder

This folder contains example screenshots and media for component documentation.

## Structure

- `ai-usage-modes/` - Screenshots for AI usage modes (Focused, Supportive, Integrated, One AI Experience)
- Component-specific folders can be added here as needed (e.g., `input/`, `tab/`, etc.)

## Usage

Images in this folder can be referenced in Next.js components using:
```jsx
<Image src="/examples/ai-usage-modes/focused-mode-1.png" alt="..." />
```
